## Module 1: Foundations of Cybersecurity

**Focus:** Core principles, threats, and security roles  

**What I did:**
- Applied CIA triad concepts when evaluating security scenarios  
- Practiced identifying phishing, malware, and social engineering techniques  
- Used SIEM dashboards in labs to observe simulated security events  

**Outcome:**
- Built foundational understanding of cybersecurity principles and threat landscape  

---

## Module 2: Security Risk Management

**Focus:** Security domains, frameworks, and controls  

**What I did:**
- Applied NIST Risk Management Framework (RMF) in structured scenarios  
- Evaluated different types of controls (technical, administrative, physical)  
- Practiced basic IAM concepts such as access control and least privilege  

**Outcome:**
- Developed ability to assess risks and understand how organizations manage security  

---

## Module 3: Network Security

**Focus:** Network architecture, protocols, and attack detection  

**What I did:**
- Analyzed network traffic and packet structure using Wireshark and tcpdump  
- Identified common attacks such as SYN flood and ICMP-based anomalies  
- Applied basic network hardening concepts (segmentation, MFA, firewall rules)  

**Outcome:**
- Gained practical skills in network analysis and identifying suspicious traffic  

---

## Module 4: Linux and SQL

**Focus:** System-level investigation and data analysis  

**What I did:**
- Used Linux CLI commands (grep, find, chmod) to analyze logs and manage access  
- Modified file permissions and practiced user management  
- Wrote SQL queries to filter and investigate login activity  

**Outcome:**
- Built strong foundation in system investigation and log analysis

## Module 5: Assets, Threats, and Vulnerabilities

**Focus:** Risk assessment and vulnerability management
